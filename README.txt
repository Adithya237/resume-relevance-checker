# Automated Resume Relevance Check System

An AI-powered system for automatically evaluating resumes against job descriptions, providing relevance scores and improvement feedback.

## Features

- **Resume Parsing**: Extract text from PDF and DOCX resumes and structure the information
- **JD Parsing**: Parse job descriptions to extract key requirements, skills, and qualifications
- **Relevance Scoring**: Calculate relevance scores using both hard matching (keywords) and soft matching (semantic similarity)
- **Gap Analysis**: Identify missing skills, qualifications, and experience
- **Personalized Feedback**: Generate actionable suggestions for resume improvement
- **Web Dashboard**: Streamlit-based interface for easy interaction
